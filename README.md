# UniSphere

UniSphere is an Android UI concept built using Jetpack Compose.  
The project focuses on clean UI structure, modern design patterns, and reusable composable components.

It is intended as a UI reference and a Compose-based starting point for Android applications.

## Tech Stack

- Kotlin
- Jetpack Compose
- Android Studio

---

## Scope

This project is UI-focused only.  
No backend integration or complex business logic is included.

---

## Getting Started

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run on an emulator or physical device
